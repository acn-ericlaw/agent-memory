## Decisions Made

- Title carries both start and end as full ISO 8601 with ms — unambiguous session boundaries, aids conflict tracing.
- Migrated sessions with a single known timestamp: repeat it for both start and end.
- Retroactive sessions (mercury-composable): used `T10:00:00.000Z` and `T14:00:00.000Z` as stand-in times since originals were not recorded.
