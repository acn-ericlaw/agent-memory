helper the agent/vendor/CI invokes; the tool runs nothing itself). This **reverses** the deliberate
"no sync script, agent acts out the recipe" choice — evidence showed it had a real reliability cost.

- For each `agent-skills/<name>/SKILL.md` it (re)writes all 5 vendor adapters and **prunes orphans it
  generated** — *signature-guarded* (only removes files containing our pointer signature), so a
  hand-authored vendor skill is never deleted. Writes only gitignored adapter dirs; idempotent.
- `SKILL.md` carries `provenance: agent-memory-builtin` + a "this IS the operation; don't hunt for a
  command, run the script" body, so an agent finds it **by description** and runs it.
